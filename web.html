Let's try making a simple website using HTML, CSS, and JavaScript!

  • Hello World - http://codepen.io/seanhelvey/pen/OVqpXg

  • Things to do with Hello World:

    • Change text and background color
      • CSS:
        html { color: purple; background-color: lightblue; }
    • Make body into h1 and h2
      • HTML:
        <h1>Hello, World!</h1>
        <h2>Here are a few of my favorite things:</h2>
    • Add list of favorite things
      • HTML:
        <ul>
          <li>Cheese</li>
          <li>Surfing</li>
          <li>Dogs</li>
        </ul>
    • Style list to have different background color
      • CSS:
        ul { background-color: yellow; }
    • Add button with onclick event and javascript function
      • HTML:
        <button onclick="changeName()">Click me!</button>

        JavaScript:
        function changeName(){
          alert("test");
        }
    • Put span around world and change color (explain css # for select by name)
      • HTML:
        <span id="name">World</span>

        CSS:
        #name { color: red; }
    • Write function to set new name
      • HTML:
        <input id="myinput"></input>

        JavaScript:
        function changeName(){
          var name = document.getElementById("name");
          var myinput = document.getElementById("myinput");
          name.innerHTML = myinput.value;
        }
